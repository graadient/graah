# GRAAH

Source for `graah.se`, built with Vite, React, Tailwind CSS, and Hero UI.

## Development

```sh
npm install
npm run dev
```

## Checks

```sh
npm run verify
```

## Deployment

GitHub Pages deploys `dist/` from the `main` branch through
`.github/workflows/pages.yml`. The custom domain is set by `public/CNAME`.
